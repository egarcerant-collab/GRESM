# **App Name**: Audit Logger

## Core Features:

- Form Generation: Dynamically generate audit forms based on predefined templates.
- Data Input Fields: Provide a variety of input fields including text areas, dropdowns, date pickers, and document upload, specifically catering to the fields you provided (Alice Plata, patient name, document type, etc.)
- Local Data Storage: Store form data locally using the file system.
- Activity Log: Maintain an audit log of all actions performed on the forms, to enhance quality assurance. An LLM tool will summarize key logs in a paragraph that can be exported as a document.
- Reporting: Generate summary reports of the audit activities.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) for professionalism and reliability.
- Background color: Light gray (#ECEFF1) for a clean, neutral backdrop.
- Accent color: Teal (#009688) to highlight important actions and information.
- Body and headline font: 'PT Sans', a humanist sans-serif that offers a blend of modern aesthetics and subtle warmth, suitable for both headings and content text.
- Use simple, clear icons from a library like Material Design Icons to represent actions and data types.
- Employ a responsive layout that adapts to different screen sizes, ensuring usability across devices.
- Use subtle transitions to provide feedback and enhance user experience, e.g., when a form is submitted or data is saved.
